# Valheim Vehicle Marker

Simple Valheim mod that adds markers for all ships and carts to the map.

## Configuration

Configure via config file:

```toml
[Types]

## Show ships on the map.
# Setting type: Boolean
# Default value: true
Show ships = true

## Show carts on the map.
# Setting type: Boolean
# Default value: true
Show carts = true
```
